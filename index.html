SkillBridge API

Backend Service for SkillBridge Platform

Welcome to the SkillBridge backend API. This service provides the RESTful API endpoints for the SkillBridge platform.

Available API Endpoints

GET /api/health

Health check endpoint to verify API status

GET /api/test

Test endpoint to verify basic functionality

GET /api/db-test

Database connection test endpoint

POST /api/auth/register

User registration endpoint

POST /api/auth/login

User authentication endpoint

GET /api/courses

Retrieve available courses

GET /api/mentors

Retrieve mentor information

Note: This is the backend API service. For the full SkillBridge experience, please visit our frontend application and configure it to connect to this API endpoint.
